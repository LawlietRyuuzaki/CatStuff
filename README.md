# CatStuff
Random stuff that may end up to be fun.

# How to compile
It should compile on every platform as long as you have :

* CMake
* make
* gcc or mingw

On Windows
---------
```
cmake -G "MinGW Makefiles"
make
CatStuff.exe
```

On Linux
-------
```
cmake -G "Unix Makefiles"
make
./CatStuff
```
